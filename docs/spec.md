# מפרט טכני — סוכן טלגרם אישי

## סקירה כללית

סוכן טלגרם אישי בארכיטקטורה כפולה:
- **רכיב 1 — Bot API:** ממשק צ'אט יומיומי לניתוב הודעות, ניהול קבוצות, תזכורות, יומן ואנשי קשר
- **רכיב 2 — Userbot:** סריקה יומית של צ'אטים וסידור תיקיות אוטומטי

**משתמש:** יחיד בלבד

**שפה:** עברית + אנגלית

**שפת תכנות:** פייתון (עדיף לאקוסיסטם טלגרם ותמלול)

**אירוח:** Railway / Render — הפתרון הזול והפשוט ביותר

---

## רכיב 1 — Bot API

### 1.1 ניתוב הודעות לקבוצות

**תיאור:**
הבוט מקבל הודעה בצ'אט פרטי ומנתב אותה לקבוצה הנכונה.

**פורמט קלט:**
- שפה חופשית: "תשלח לקבוצת עבודה: לבדוק חוזה"
- פקודה: "/עבודה לבדוק חוזה"
- הודעה קולית עם הוראת ניתוב

**סוגי תוכן נתמכים:**
- טקסט
- תמונות
- קבצים
- קישורים
- הודעות קוליות — תמלול לטקסט או העברה כקובץ קולי, תלוי בהוראה

**זיהוי קבוצה:**
- LLM (Claude API) מנתח את ההודעה ומזהה את שם הקבוצה
- אם הבוט לא בטוח — מציג כפתורים (inline buttons) עם שמות הקבוצות לבחירה
- הודעות נשלחות בשם הבוט (לא בשם המשתמש)

**מחיקת הודעות:**
- מחיקה בפקודה ידנית בלבד
- לדוגמה: "מחק את ההודעה האחרונה ששלחתי"

### 1.2 ניהול קבוצות

**מגרסה ראשונה — פקודות:**
- הוספת קבוצה חדשה
- מחיקת קבוצה
- הצגת רשימת קבוצות
- צפייה בתוכן של קבוצה ספציפית

**ניהול דרך:** פקודות בצ'אט עם הבוט

### 1.3 תמלול הודעות קוליות

**מקורות:** טלפון + מחשב (Windows)

**התנהגות:**
- הבוט מתמלל את ההודעה הקולית
- מזהה הוראות מתוך הטקסט המתומלל (ניתוב, תזכורות, אירועים)
- לפי הוראה — מתמלל לטקסט ומעביר, או מעביר כקובץ קולי

### 1.4 סיכום חכם

**לפי דרישה בלבד** — לא אוטומטי.

כשנשאל, הבוט קורא תוכן מקבוצה ספציפית ומייצר סיכום באמצעות LLM.

לדוגמה: "סכם לי את קבוצת עבודה"

### 1.5 לוג היסטוריה

הבוט שומר לוג של כל שליחה: מה נשלח, לאיזו קבוצה, מתי. ניתן לצפות בלוג דרך פקודה בצ'אט.

### 1.6 תזכורות

הגדרה: שפה חופשית, פקודה, או הודעה קולית. ניהול: הצגה ומחיקה.

### 1.7 חיבור ל-Google Calendar

קריאה + כתיבה. תומך בכמה יומנים, יומן ברירת מחדל, הזמנת אנשי קשר לפי כינוי, ותזכורת ברירת מחדל שעה לפני האירוע.

### 1.8 ספר אנשי קשר

מיפוי כינויים לכתובות מייל לצורך הזמנות ליומן. ניתן להוסיף דרך הבוט או דרך קובץ קונפיגורציה.

### 1.9 פקודת עזרה

`/help` או "עזרה" מציגה את כל היכולות של הבוט.

### 1.10 התאוששות מתקלות

אם השרת נופל — הבוט מנסה להתחבר מחדש אוטומטית.

## רכיב 2 — Userbot

### 2.1 סריקת צ'אטים יומית

פעם ביום, סורק את כל הצ'אטים בחשבון הטלגרם, מזהה צ'אטים שלא קשורים לקבוצות הבוט, ומעביר אותם לתיקיה בשם "שאר הצ'אטים".

טכנולוגיה: Telethon. רכיב עצמאי לחלוטין מהבוט.
